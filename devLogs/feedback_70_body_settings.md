For the Export Configuration, change the settings for the table body to:

* Empty Slot Background Color
* Empty Slot Font Size
* Empty Slot Font Color

* Name Font Size (for course name, Office Hours Label, and Meetings Label)
* Name Font Color (for course name, Office Hours Label, and Meetings Label)

* Instructor Name Font Color (unique for Course Table)
* Instructor Name Font Color (unique for Course Table)

* Time Range Font Size (this is for the ones writen on each of the course / meetings / office hours / empty slots)
* Time Range Font Color (this is for the ones writen on each of the course / meetings / office hours / empty slots)

* Time Label Font Size (this is the time label on the left size of the table)
* Time Label Font Color (this is the time label on the left size of the table)

* Weekdays offset (a vertical offset for the vetical location of the week days text)
* Weekdays font color

Also, for the meetings, in the faculty schedule, remove the Meetings text, the name of the meeting should be clear enough to show that it is a meeting.
